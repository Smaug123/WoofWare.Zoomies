# Interactive Demos

These animated demos show WoofWare.Zoomies components in action. Each demo loops automatically, demonstrating keyboard interactions with the TUI framework.

## Checkbox Toggle

Demonstrates keyboard navigation between checkboxes and toggling their state with the Space key.

![Checkbox Toggle Demo](../../output/demos/checkbox-toggle.gif)

**Interactions shown:**
- Tab to navigate between focusable elements
- Space to toggle checkbox state
- Visual feedback for focus and checked state
